# **Overview**
An implementation of a FIFO matching engine in C++ 14
# **Requirements**
A chosen compiler e.g. g++

# **Installation**
```
git clone https://github.com/JadoreThompson/cpp-order-matcher.git

cd cpp-order-matcher

# Compiler - if you're using g++
g++ *.cpp -o program.exe 

# Run
program
```
